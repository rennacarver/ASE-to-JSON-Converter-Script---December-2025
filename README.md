# ASE-to-JSON-Converter-Script---December-2025
Converter for ASE Files to JSON

Steps to Use:

1. Place your file to convert in the same directory as convert-ase.js
2. Open a terminal inside that folder
3. run npm install
4. Rename the inputPath const inputPath = 'yourfilenmae"
5. Save the file
6. Run the script using the terminal 'node convert-ase.js"
7. The terminal will show a success message if it worked
